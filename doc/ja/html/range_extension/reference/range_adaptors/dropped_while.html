dropped_while

dropped_whileは、ベースとなるRangeの先頭から述語を満たす間の要素を除いたRangeを返す。

構文

コード

パイプスタイル

rng | boost::adaptors::dropped_while(pred)

regular演算子

rng |+ boost::adaptors::dropped_while(pred)

関数スタイル

boost::adaptors::drop_while(rng, pred)

  • 事前条件: predは、参照した要素を書き換えてはならない。
  • Rangeカテゴリ: Single Pass Range
  • 戻り値型: ...
  • 返されるRangeカテゴリ: rngのRangeカテゴリ
ヘッダ 

#include <boost/range/adaptor/dropped_while.hpp>

関連ヘッダ 

#include <boost/range/regular.hpp>

例 

#include <boost/range/adaptor/dropped_while.hpp>
#include <boost/range/algorithm/for_each.hpp>
#include <boost/assign/list_of.hpp>
#include <iostream>
#include <vector>

void disp(int x) { std::cout << x << ' '; }
bool is_even(int x) { return x % 2 == 0; }

using namespace boost::adaptors;

int main()
{
    const std::vector<int> v = boost::assign::list_of(2)(4)(6)(1)(3);

    boost::for_each(v | dropped_while(is_even), disp);
}

これは以下の結果を出力する:
