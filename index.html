History and Acknowledgement

Boost.Range is a collection of concepts and utilities, range-based algorithms, as well as range adaptors that allow for efficient and expressive code.

Using Boost.Range inplace of the standard library alternatives results in more readable code and in many cases greater efficiency.

This library is enhanced version of Boost Range Library. It has powerful Range adaptors and utilities for Range programming. For example, using this library, we can write quick sort as the following: 

typedef
    any_range<int, forward_traversal_tag, int&, std::ptrdiff_t>
range;

range quick_sort(range rng)
{
    if (!rng)
        return rng;
    else {
        int& x = rng | front;
        range xs = rng | dropped(1);

        return
            join(
                join(
                    quick_sort(xs |+ filtered(_1 < x)),
                    single(x)
                ),
                quick_sort(xs |+ filtered(_1 >= x))
            );
    }
}

std::vector<int> v = boost::assign::list_of(3)(2)(5)(4)(6)(1);
std::cout << quick_sort(range(v)) << std::endl;
