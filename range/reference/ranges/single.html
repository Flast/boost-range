single

Prototype 

template <class T>
iterator_range<const T*> single(const T& x);

template <class T>
iterator_range<T*> single(T& x);

Namespace

boost or boost::range.

Description

single makes a Random Access Range which delivers a range presentation of one object.

Header 

<boost/range/single.hpp>

or 

<boost/range/range/single.hpp>

Returned Range Category

Random Access Range

Example 

#include <iostream>
#include <boost/range/single.hpp>
#include <boost/range/algorithm/for_each.hpp>

void disp(int x) { std::cout << x << std::endl; }

int main()
{
    boost::for_each(boost::single(3), disp);
}

This would produce the output: 

3
